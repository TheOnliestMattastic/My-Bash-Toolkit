My Bash Toolkit

A curated collection of automation scripts for Linux systems

📖 Overview

This repository showcases a set of Bash scripts I’ve written to automate everyday tasks, streamline workflows, and demonstrate practical scripting skills. Each script is self-contained, documented, and easy to adapt.

🛠️ Scripts Included

  • installFlatpakAndDnf.sh — Installs a curated list of dnf and Flatpak packages for Fedora/Nobara systems.
  • mountMyOneDrive.sh — Mounts a OneDrive remote via rclone, with status notifications.
  • myLoginScript.sh — Applies an OpenRGB lighting profile and displays a fun fortune message at login.
  • scheduleIcheats.sh — Automates keybindings in a game console using xdotool.

🚀 Getting Started

  1. Clone the repository: 
    git clone https://github.com/TheOnliestMattastic/My-Bash-Toolkit.git
cd My-Bash-Toolkit
  2. Make a script executable: 
    chmod +x script-name.sh
  3. Run it: 
    ./script-name.sh

Note: Some scripts require dependencies like rclone, xdotool, or OpenRGB. See script headers for details.

💡 Examples

Mount OneDrive:

./mountMyOneDrive.sh

Install packages:

./installFlatpakAndDnf.sh
